Memory Match

How to Play

 Each tile has two faces. One is shared between all tiles and is on display by default. The other is revealed once a tile has been clicked and has only one matching pair. You can only flip two tiles at a time at which point the game will check to see if they match. If you reveal a matching pair, both tiles will remain flipped, otherwise they will both go back to to the default image. You win once all tiles have been revealed.
